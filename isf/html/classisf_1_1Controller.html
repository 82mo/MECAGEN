Detailed Description

template<typename MetaParam, typename Param, typename State>
class isf::Controller< MetaParam, Param, State >

The Controller Class.

The controller managed the execution of a model. Controller is an abstraction of the controller used in the framework while Controller_Impl is the real implementation.
