Detailed Description

template<typename MetaParam, typename Param, typename ParamDevice, typename State, typename StateDevice, typename Model>
class isf::Controller_Impl_Host_Device< MetaParam, Param, ParamDevice, State, StateDevice, Model >

The Controller_Impl_Host_Device Class.

This class inherits from the Controller_Impl and it aims at implementing the "loadDeviceState" and "loadDeviceParam" methods which copy the State and Param objects from the host backend to the device backend in the Host-Device mode of the framework. This class is the one implemented in the client code.
