Detailed Description

template<typename T>
class isf::EventManager< T >

EventManager class.

The EventManager class is a templated class which contains the synchronization mechanism designed in ISF. It contains a internal stack of event: a std::deque< Event<T>* > that is filled by the notifying thread via the send() method and read by the event manager's owner thread via the read() method. The synchronization is assured by a std::mutex variable that the sending and reading threads acquire and release when they use the deque.
