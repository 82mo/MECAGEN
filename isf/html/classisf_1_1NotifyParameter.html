Detailed Description

NotifyParameter class.

The NotifyParameter class contains the pair of parameters that are passed in an event send by a broker to a consumer.

Parameters
[in]brokerIDThe broker ID, stored as pointer of ProducerFactory
[in]stepThe time step associated with the notification
[in]errorCodeThe errorCode of the notification, used optionally
