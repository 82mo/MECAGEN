Detailed Description

template<typename MetaParam, typename State, typename StateDevice>
class isf::Producer< MetaParam, State, StateDevice >

The Producer Class.

The Producer class is a templated class that is the top level of the producer concept. It is the parent class of the ISF-provided Producer_Impl_Host_Only and Producer_Impl_Host_Device classes and the grand-parent of the client-provided producer class. It implements the mechanism of execution of the producers and the control of the broker swapping.
